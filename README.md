# Hangman Game

## Project Description
This is a simple text-based Hangman Game developed using Python. The player has to guess the hidden word one letter at a time. The game provides six attempts to guess the correct word.

## Features
- Random word selection
- User-friendly interface
- Input validation
- Correct and wrong guess detection
- Limited number of attempts
- Winning and losing messages

## Technologies Used
- Python 3
- VS Code

## Concepts Used
- Random Module
- Lists
- Strings
- While Loop
- If-Else Statements
- Functions
- User Input

## How to Run
1. Open the project in VS Code.
2. Run the `hangman.py` file.
3. Enter one letter at a time.
4. Guess the word before the attempts finish.
